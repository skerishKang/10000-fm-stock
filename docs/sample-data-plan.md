# 샘플 데이터 설계

## 배경

MVP 화면 검증을 위해 각 엔티티별로 충분한 양과 다양한 조합의 샘플 데이터가 필요하다. 본 문서는 샘플 데이터의 규모, 구성 방향, 각 엔티티별 세부 기준을 정의한다.

## 샘플 데이터 규모

| 엔티티 | 목표 건수 | 비고 |
|--------|----------|------|
| experts | 5~10명 | 가명의 애널리스트/투자자 |
| sources | 10~20개 | 유튜브, 리포트, IR/공시 |
| segments | 20~40개 | 발췌 구문 단위 |
| claims | 30~60개 | 주장 단위, 결과/방향성 다양 |
| evaluations | 20~40개 | 수익률/알파 다양한 조합 |
| knowledge_notes | 30~60개 | 지식/맥락 노트 |

각 엔티티의 목표 건수는 MVP 화면에서 리스트, 상세, 검색, 통계 화면이 정상 동작함을 확인할 수 있을 정도로 설정하였다.

## Expert 구성

### 대상자 유형 (6종)
1. **반도체 애널리스트** (가명: 김석민) — 10년차, 대형 증권사, 반도체 커버
2. **2차전지 애널리스트** (가명: 이에코) — 8년차, 중형 증권사, 배터리 커버
3. **경제방송 출연자** (가명: 박마켓) — 15년차, 방송인, 시황 분석
4. **유튜브 투자 채널** (가명: 주식하는사람) — 구독자 10만, 개인투자자
5. **장기 투자자** (가명: 최배당) — 20년차, 가치투자, 배당주 중심
6. **리포트 작성자** (가명: 정애널) — 5년차, 리서치센터, 신규 커버

### 보유 속성
- 이름 (가명), 소속, 경력 연차, 전문 분야, 프로필 이미지(더미)

## Source 구성

### 유형별 분포
| 유형 | 건수 | 예시 |
|------|------|------|
| 유튜브 영상 | 5~10개 | 종목 분석, 시황, 산업 전망 |
| 기관 리포트 | 5~10개 | 반도체/2차전지/조선/바이오/AI |
| IR 자료/공시 | 1~2개 | 실적 발표, M&A 공시 |

### 보유 속성
- 제목, 유형, URL (더미 URL 허용), 발행일, expert 참조, 전문(raw text, 더미)

### 더미 URL 규칙
- `https://youtube.example.com/watch/{id}`
- `https://report.example.com/{broker}/{date}/{title}`
- `https://dart.example.com/{corp}/{date}/{title}`

## Segment 구성

- 각 source 당 1~5개 segment 할당
- segment 길이: 1~5문장
- source의 핵심 주장 단위로 분할
- 최소 20개, 최대 40개

## Claim 구성

### 결과 (judgment) 분포
| 결과 | 비율 | 건수 (30~60 기준) |
|------|------|-------------------|
| hit | ~30% | 9~18건 |
| partial_hit | ~20% | 6~12건 |
| miss | ~20% | 6~12건 |
| invalid | ~10% | 3~6건 |
| pending | ~20% | 6~12건 |

### 방향성 (direction) 분포
| 방향성 | 비율 | 설명 |
|--------|------|------|
| bullish | ~30% | 상승/매수 관점 |
| bearish | ~20% | 하락/매도 관점 |
| neutral | ~15% | 중립 관점 |
| mixed | ~15% | 혼합 관점 |
| educational_only | ~20% | 정보 제공, 방향성 없음 |

### 종목 커버리지
| 섹터 | 종목 예시 | claim 수 |
|------|----------|---------|
| 반도체 | 삼성전자, SK하이닉스, 한미반도체 | 8~16건 |
| 2차전지 | LG에너지솔루션, 포스코퓨처엠, 에코프로비엠 | 6~12건 |
| 조선 | HD현대중공업, 삼성중공업, 한화오션 | 4~8건 |
| 바이오 | 삼성바이오로직스, 셀트리온, 알테오젠 | 4~8건 |
| AI | NAVER, 카카오, 솔루엔 | 4~8건 |
| 기타/혼합 | 시황, 지수, 섹터 전반 | 4~8건 |

### 속성
- claim_text, judgment, direction, expert 참조, segment 참조, date, ticker, sector

### 판정별 테스트 가능성
- **hit**: 실제 주가 상승으로 검증된 사례 → 화면에서 "정확" 태그 확인
- **partial_hit**: 부분적 실현 (시점/목표가 차이) → 화면에서 "부분" 태그 확인
- **miss**: 예측과 다른 방향 → 화면에서 "오류" 태그 및 상세 비교
- **invalid**: 확인 불가/모호한 진술 → 화면에서 "판정 불가" 상태 확인
- **pending**: 아직 평가되지 않은 최근 진술 → 화면에서 "대기" 상태 확인

## Evaluation 구성

### 수익률과 알파의 다양한 조합
| 시나리오 | 예시 | 건수 |
|---------|------|------|
| +수익률 +알파 | 15% 수익, 벤치마크 대비 +8% | 4~8건 |
| +수익률 -알파 | 5% 수익, 벤치마크 대비 -3% | 4~8건 |
| -수익률 +알파 | -10% 손실, 벤치마크 대비 +5% | 4~8건 |
| -수익률 -알파 | -8% 손실, 벤치마크 대비 -6% | 4~8건 |
| 0 수익률 근접 | ±2% 이내, 알파 근접 | 4~8건 |

### 보유 속성
- claim 참조, 수익률(%), 알파(%), 기간(1개월/3개월/6개월/1년), 평가일

### 주제 분류
| 주제 | 설명 | note 수 |

| DRAM 사이클 | 메모리 반도체 수급/가격 사이클 | 3~6건 |

| 2차전지 양극재 | 양극재 가격/수급 동향 | 3~6건 |
| 조선 수주잔고 | 글로벌 조선 수주/발주 동향 | 3~6건 |
| 바이오 CMO | 바이오 의약품 위탁생산 수요 | 3~6건 |

### 보유 속성
- title, content, category, source 참조, created_at

## 화면 검증 가능성

각 화면별 검증 시나리오는 다음과 같다.

### 리스트 화면
- expert 목록 5~10명 표시 확인
- source 목록 10~20개 표시 확인
- claim 목록 30~60개 표시 확인 (페이지네이션)
- knowledge note 목록 30~60개 표시 확인 (페이지네이션)

### 상세 화면
- expert 상세: 프로필, 관련 source, claim, evaluation
- claim 상세: claim_text, 판정, 방향성, source 링크, evaluation
- evaluation 상세: 수익률, 알파, 차트 (더미 데이터)

### 검색/필터 화면
- expert 이름/소속 검색
- claim 결과(hit/miss 등) 필터
- claim 방향성(bullish/bearish) 필터
- 섹터/종목 필터

### 통계 화면
- expert별 hit rate 분포
- 방향성별 판정 분포
- 섹터별 claim 분포
- evaluation 수익률 분포 (histogram)
