## 1. 개요

본 문서는 FM 발언 검증 시스템에서 발언(claim) 단위의 수익률, 초과수익률(알파), 적중률(hit rate), 목표가 도달률 등을 계산하고 평가하는 기준을 종합적으로 정의합니다.

### 1.1 목적

- FM 발언의 객관적이고 일관된 성과 측정
- 발언자 간 공정한 비교를 위한 표준화된 평가 프레임워크 제공
- 투자자에게 유의미한 성과 지표를 투명하게 공개
- 극단값(outlier) 왜곡을 방지한 통계적 보정 체계 마련

### 1.2 설계 원칙

1. **투명성 (Transparency)**: 모든 계산식과 기준을 명확히 문서화
2. **일관성 (Consistency)**: 동일한 조건에서는 항상 동일한 결과 도출
3. **공정성 (Fairness)**: 표본 수 차이, 극단값 등 왜곡 요인 보정
4. **확장성 (Scalability)**: 새로운 자산군, 기간, 지표 추가 용이
5. **실용성 (Practicality)**: 실제 투자 의사결정에 도움되는 지표 우선

## 3. 필요 정보 필드

각 claim을 평가하기 위해 다음 정보가 필요합니다:

### 3.1 입력 필드

| 필드 | 타입 | 설명 | 필수 | 예시 |
|------|------|------|------|------|
| 발언일 | date | 발언이 이루어진 날짜 | Y | 2024-01-15 |
| 기준일 | date | 기준가를 결정하는 날짜 | Y | 2024-01-15 |
| 기준가 | number | 기준일의 종가 (또는 해당 시점 가격) | Y | 50,000원 |
| 검증일 | date | 성과를 검증하는 날짜 | N | 2024-07-15 |
| 검증가 | number | 검증일의 종가 (또는 해당 시점 가격) | N | 65,000원 |
| 방향성 | enum | 발언의 방향성 분류 | Y | bullish |
| 목표가 | number | 발언자가 제시한 목표 가격 | N | 80,000원 |
| 목표기간 | string | 목표 달성 예상 기간 | N | 6개월 |
| 벤치마크 | string | 초과수익률 계산용 기준 지수 | N | KOSPI |

### 3.2 기준가 결정 원칙

1. **기준일이 발언일과 동일한 경우**: 발언일 종가 사용 (장중 발언 시 장중 가격 사용 가능)
2. **기준일이 발언일과 다른 경우**: 기준일 종가 사용
3. **장중 발언**: 장중 가격을 기준가로 사용할 수 있으며, 이 경우 별도 표시
4. **재무제표 기준 발언**: 발언일 이후 최초 공시일 종가를 기준가로 사용

### 3.3 검증가 결정 원칙

1. **검증일 종가**: 기본값
2. **목표가 도달 검증**: 검증 기간 내 목표가 도달 여부는 일중 고가/저가 기준으로 판단 가능
3. **분할/합병 등**: 과거 가격은 분할/합병 조정 반영

## 4. 방향성 분류

모든 claim은 다음 5가지 방향성 중 하나로 분류됩니다.

### 4.1 방향성 정의

| 분류 | 코드 | 설명 | 검증 여부 |
|------|------|------|-----------|
| 상승 | bullish | 가격이 오를 것이라는 예측 | 검증 |
| 하락 | bearish | 가격이 내릴 것이라는 예측 | 검증 |
| 중립 | neutral | 가격이 유지되거나 변동 없을 것이라는 예측 | 검증 (범위 이탈 시 miss) |
| 혼합 | mixed | 상승과 하락 요소가 혼재된 발언 | 검증 (상세 조건 필요) |
| 교육용 | educational_only | 시장 설명이나 교육적 목적, 예측 아님 | 검증 제외 |

### 4.2 분류 기준

- **bullish**: "산다", "매수", "상승", "급등", "간다", "up", "long" 등의 표현 포함
- **bearish**: "판다", "매도", "하락", "조정", "폭락", "down", "short" 등의 표현 포함
- **neutral**: "횡보", "박스권", "hold", "관망" 등의 표현 포함
- **mixed**: "~하지만 ~하다", "단기~장기~" 등 상반된 전망 포함
- **educational_only**: 시장 원리 설명, 과거 데이터 분석, 조건부 가정

### 4.3 모호한 발언 처리

모호한 발언은 로 분류하여 검증에서 제외합니다. 기준:
- 방향성이 명확하지 않은 경우 ("~수도 있다", "~가능성")
- 조건이 너무 많아 실제 예측으로 보기 어려운 경우
- "~아니면 ~"처럼 양방향을 모두 열어둔 경우

## 5. 수익률 계산

### 5.1 기본 수익률

발언의 기본 수익률은 다음과 같이 계산합니다:



- : 기준가
- : 검증가

### 5.2 수익률 표시

- 소수점 둘째 자리까지 표시 (예: +12.34%, -5.67%)
- 양수는 상승, 음수는 하락 의미
- 0%는 기준가와 동일

### 5.3 방향성별 수익률 해석

방향성에 따라 수익률의 의미가 다릅니다:

- **bullish**: 양수 수익률 = 예측 성공 (방향 일치), 음수 수익률 = 예측 실패
- **bearish**: 음수 수익률 = 예측 성공 (방향 일치), 양수 수익률 = 예측 실패
- **neutral**: 절대값 수익률이 작을수록 예측 성공
- **mixed**: 방향성 가중치를 적용한 복합 수익률 계산

## 6. 초과수익률(알파) 계산

### 6.1 알파 정의

초과수익률(알파)은 발언 수익률에서 동일 기간 벤치마크 수익률을 차감하여 계산합니다:



- : 해당 자산의 수익률
- : 동일 기간 벤치마크 지수 수익률

### 6.2 벤치마크 지정 규칙

| 자산군 | 기본 벤치마크 | 설명 |
|--------|--------------|------|
| 코스피 종목 | KOSPI | 한국 대표 지수 |
| 코스닥 종목 | KOSDAQ | 코스닥 지수 |
| 해외 주식 | S&P 500 / NASDAQ | 시장별 선택 |
| 원자재 | CRB / 해당 원자재 지수 | 원자재 벤치마크 |
| 암호화폐 | BTC / ETH | 시장 대표 코인 |
| 기타 | 발언자 지정 벤치마크 | 명시된 경우 우선 |

### 6.3 알파 해석

- **양수 알파 (+α)**: 시장보다 outperformed, 발언자가 시장을 이겼음
- **음수 알파 (-α)**: 시장보다 underperformed, 발언자가 시장에 졌음
- **알파 = 0**: 시장 수익률과 동일

## 7. 검증 기간

### 7.1 기본 검증 기간

모든 claim에 대해 다음 4가지 기본 기간으로 수익률을 계산합니다:

1. **1M (1개월)**: 기준일 + 1개월
2. **3M (3개월)**: 기준일 + 3개월
3. **6M (6개월)**: 기준일 + 6개월 (기본값)
4. **12M (12개월)**: 기준일 + 12개월

### 7.2 목표기간 우선 원칙

발언자가 목표기간을 명시한 경우, 해당 기간을 우선 검증 기간으로 사용합니다:

- 목표기간 = "3개월" → 3M을 주 검증 기간으로 사용
- 목표기간 = "1년" → 12M을 주 검증 기간으로 사용
- 나머지 기간은 부수적으로 계산

### 7.3 기본값

목표기간이 명시되지 않은 경우, 기본 검증 기간은 **6개월(6M)** 로 설정합니다.

## 9. Hit 기준 상세

### 9.1 Bullish Hit 기준

bullish 발언이 hit로 판정되기 위한 조건:

1. **기본 조건**: 검증가 > 기준가
2. **유의미 상승**: 수익률이 최소 +2% 이상 (노이즈 구간 제외)
3. **목표가 도달**: 검증 기간 내 목표가를 한 번이라도 도달 (종가 기준 또는 일중 기준)
4. **시장 대비**: 시장 하락장에서 방어했을 경우 hit로 인정 가능



### 9.2 Bearish Hit 기준

bearish 발언이 hit로 판정되기 위한 조건:

1. **기본 조건**: 검증가 < 기준가
2. **유의미 하락**: 수익률이 최소 -2% 이하
3. **목표가 도달**: 검증 기간 내 목표가 도달
4. **시장 대비 방어**: 시장 상승장에서 상승률이 시장보다 낮았을 경우 인정 가능



### 9.3 Neutral Hit 기준

neutral 발언이 hit로 판정되기 위한 조건:

1. 검증가가 기준가 대비 ±3% 이내에서 형성
2. 목표 범위가 명시된 경우, 해당 범위 내에 검증가가 위치

## 12. 통계 보정

### 12.1 평균과 중앙값 병행

### 12.2 표본 수 보정 (Sample Size Correction)

1. **최소 표본 수**: 통계에 포함되기 위한 최소 claim 수는 5개로 설정

3. **신뢰 구간**: 표본 수가 적을수록 신뢰 구간이 넓어짐을 표시

## 13. 종합 점수 vs 다각도 랭킹

### 13.1 초기 방침: 다각도 랭킹

초기에는 종합 점수(composite score) 없이 여러 기준으로 랭킹을 제공합니다.
종합 점수는 충분한 데이터가 축적되고 가중치에 대한 합의가 이루어진 후 도입합니다.

### 13.2 제공되는 랭킹 기준

1. **수익률 랭킹**: 평균 수익률 순
2. **중앙값 수익률 랭킹**: 중앙값 수익률 순
3. **적중률 랭킹**: hit rate 순
4. **알파 랭킹**: 초과수익률 순
5. **목표가 도달률 랭킹**: 목표 달성 비율 순
6. **표본 수 랭킹**: 가장 많은 발언을 한 순

### 13.3 랭킹 표시 규칙

- 모든 랭킹에는 발언자 이름, 값, 표본 수를 반드시 표시
- 표본 수가 최소 기준(5개) 미만인 발언자는 별도 섹션에 표시하거나 "표본 부족"으로 표기
- 1위부터 10위까지 기본 표시, 더보기 옵션 제공

## 17. 향후 확장

### 17.1 단계적 도입 로드맵

**Phase 1 (현재)**: 기본 수익률, 적중률, 알파 계산 기준 수립
**Phase 2**: 자동 검증 파이프라인 구축 (정기적인 시세 수집 및 검증)
**Phase 3**: 대시보드 및 랭킹 페이지 구축
**Phase 4**: 종합 점수(composite score) 도입 검토
**Phase 5**: 머신러닝 기반 발언 품질 평가 모델 도입

### 17.2 추가 고려 지표

- **Sharpe Ratio**: 수익률 대비 변동성 고려
- **Information Ratio**: 초과수익률 대비 트래킹 에러
- **Max Drawdown**: 최대 손실 폭
- **Win/Loss Ratio**: 평균 이익 / 평균 손실 비율
- **Profit Factor**: 총 이익 / 총 손실
- **Recovery Factor**: 총 수익 / Max Drawdown
- **Calmar Ratio**: 연간 수익률 / Max Drawdown

### 17.3 고도화 방향

- **발언 난이도 가중치**: 시장 상황(변동성, 추세 강도)에 따른 가중치 조정
- **시계열 가중치**: 최근 발언에 더 높은 가중치 부여
- **군집 효과 보정**: 동일 시점 동일 방향 발언의 군집 효과 제거
- **크로스 검증**: 발언자의 전체 예측 대비 특정 종목 편향 분석
