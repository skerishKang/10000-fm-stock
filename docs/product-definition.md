# 제품 정의서: 10000 FM Stock

> **버전:** 1.0  
> **작성일:** 2026-05-12  
> **상태:** 초기 정의 (MVP)

---

## 1. 프로젝트 개요

**10000 FM Stock**은 주식 전문가와 애널리스트의 공개 발언을 기록·검증하고, 산업 및 종목에 관한 지식을 체계적으로 축적하는 플랫폼이다.

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 10000 FM Stock |
| 유형 | 전문가 발언 검증 + 지식 베이스 플랫폼 |
| 대상 사용자 | 개인 투자자, 예비 투자자 |
| 핵심 가치 | 데이터 기반 투자 판단 지원, 정보 비대칭 해소 |

---

## 2. 핵심 목적

### 2.1. 전문가 발언 검증

전문가(주식 전문가, 애널리스트, 경제 유튜버 등)의 공개 발언을 수집하여 **데이터 기반의 수익률 및 적중률**을 산출한다.

- 특정 전문가의 과거 발언을 실제 주가 흐름과 비교
- 예측/분석의 정확도를 정량적으로 측정
- 검증 결과를 누구나 열람 가능한 형태로 제공

### 2.2. 교육용 지식 DB 축적

전문가의 분석 내용에서 핵심적인 산업 지식과 투자 논리를 추출하여 **구조화된 지식 데이터베이스**를 구축한다.

- 산업 구조 분석 (밸류체인, 경쟁 구도, 규제 환경 등)
- 기업 경쟁력 평가 (비즈니스 모델, 재무 건전성, Moat 등)
- 리스크 요인 정리 (규제 리스크, 기술 리스크, 시장 리스크 등)
- 투자 판단 프레임워크 학습 자료

---

## 3. 핵심 원칙

### 3.1. 원본은 출처, 진짜 자산은 구조화 데이터

- 모든 데이터는 원본 출처(영상, 기사, 방송 등)를 명확히 링크한다.
- 플랫폼의 진정한 가치는 원본 복제가 아닌, **정제·구조화된 데이터**에 있다.
- 구조화된 데이터는 검색, 비교, 분석이 가능한 형태로 저장한다.

### 3.2. 6개월~1년 전 자료 우선

- 단기적 hot topic보다 **시간이 지나도 가치 있는 자료**를 우선 처리한다.
- 과거 자료일수록 검증(실제 주가 vs 예측)이 용이하다.
- 신규 자료보다 아카이브 구축에 집중한다.

### 3.3. 초기 MVP 기술 스택

```
HTML + CSS + JavaScript + JSON (정적 사이트)
```

- 서버, 데이터베이스, 로그인 시스템 없이 시작한다.
- 정적 파일로도 충분히 의미 있는 서비스를 구축 가능하다.
- 추후 필요에 따라 점진적으로 확장한다.

### 3.4. DB/로그인/자동화는 나중에

**Post-MVP (v2.0+) 항목:**

- 관계형 데이터베이스 도입 (PostgreSQL 등)
- 사용자 계정 및 로그인 시스템
- 크롤링/스크래핑 자동화
- 실시간 데이터 업데이트 파이프라인
- 검색 엔진 최적화

초기에는 **수동으로 수집·정제·입력**하는 프로세스를 유지하며, 데이터 품질과 구조 설계에 집중한다.

## 4. 비목표 (Non-Goals)

| 비목표 | 설명 |
|--------|------|
| ❌ 매수/매도 추천 | 본 플랫폼은 특정 주식의 매수/매도를 추천하지 않는다. |
| ❌ 실시간 리딩방/자동매매 | 단기 시세跟进이나 자동 거래 시스템을 제공하지 않는다. |
| ❌ 원본 복제/재배포 | 전문가의 콘텐츠를 무단 복제하거나 재배포하지 않는다. 인용과 링크만 제공한다. |
| ❌ 인신공격성 표현 | 특정 전문가나 인물에 대한 인신공격적·모욕적 표현을 금지한다. |

## 6. 초기 MVP 범위

### v1.0 핵심 기능

1. **전문가 발언 카드** — 전문가별 발언 기록과 실제 주가 흐름 비교
2. **지식 카드** — 산업/종목/리스크 관련 구조화된 지식 항목
3. **검증 통계** — 전문가별 예측 적중률, 수익률 등 요약
4. **정적 사이트** — HTML/CSS/JS/JSON 기반, GitHub Pages 등에 호스팅

### v1.0 제외 기능

- 사용자 계정 및 로그인
- 데이터 자동 수집
- 검색 기능 (정적 사이트 내 간단한 JS 필터링으로 대체)
- 모바일 앱

## 10. 성공 지표 (Key Metrics)

- **데이터 수**: 구조화된 발언/지식 카드 수
- **검증 완료율**: 수집된 발언 중 검증(수익률/적중률 산출)이 완료된 비율
- **사용자 방문 수**: 정적 사이트 월간 방문자 수
- **재방문율**: 1회 이상 재방문한 사용자 비율
- **신규 제안 피드백**: 사용자로부터 받은 데이터/기능 제안 수

## 11. 라이선스 및 윤리 정책

- **데이터 라이선스**: 모든 구조화 데이터는 CC BY-NC 4.0 (비상업적 이용 허용, 출처 표시 필수)
- **출처 표시 원칙**: 모든 데이터는 원본 출처(URL, 일자, 전문가명)를 반드시 포함
- **저작권 존중**: 원본 콘텐츠를 복제하지 않으며, 인용과 요약만 제공
- **공정 사용**: 교육 및 정보 제공 목적에 한정, 상업적 재배포 금지
- **커뮤니티 가이드라인**:
  - 특정 전문가에 대한 인신공격 금지
  - 사실에 기반한 검증, 의도적 왜곡 금지
  - 검증 결과는 데이터에 기반하여 객관적으로 표시
