## 1. 개요

Ranking 화면은 사용자가 다양한 기준으로 발언, 전문가, 산업별 성과를 비교할 수 있는 종합 성과 분석 페이지입니다. 단순한 순위 표시를 넘어, 사용자가 데이터를 기반으로 전문가의 예측 능력을 평가하고 투자 판단에 참고할 수 있도록 설계되었습니다.

이 문서는 Ranking 화면의 기능적 요구사항, 데이터 구조, UI/UX 설계, API 명세를 정의합니다.

## 2. 핵심 방향

### 2.1 랭킹 = 흥미 요소 + 전문가 검증 기능

랭킹은 단순히 순위를 보여주는 것 이상의 가치를 제공해야 합니다. 사용자가 흥미를 가지고 참여할 수 있도록 게이미피케이션 요소를 도입하고, 동시에 전문가의 발언을 검증하는 신뢰할 수 있는 도구 역할을 해야 합니다.

### 2.2 단순 줄 세우기 방지, 다양한 기준 제공

하나의 종합 점수로 모든 것을 평가하는 방식은 지양합니다. 다양한 기준을 제공하여 사용자가 자신의 관심사와 투자 스타일에 맞게 랭킹을 조회할 수 있도록 합니다. 예를 들어, 단기 수익률이 높은 발언과 장기적으로 꾸준한 성과를 내는 전문가는 서로 다른 기준으로 평가되어야 합니다.

## 3. 랭킹 목적

Ranking 화면은 다음 5가지 질문에 답할 수 있도록 설계되었습니다.

### 3.1 어떤 발언이 가장 높은 수익률을 기록했는가?

과거 발언 중에서 실제 주가 수익률이 가장 높았던 발언을 확인할 수 있습니다. 단순 수익률 기준으로 정렬하여 사용자가 가장 임팩트 있는 발언을 쉽게 찾을 수 있습니다.

### 3.2 어떤 발언이 시장 대비 가장 좋은 초과수익률을 기록했는가?

동일 기간 시장 수익률(벤치마크)을 고려한 초과수익률 기준으로 발언을 평가합니다. 시장이 상승장일 때 단순 수익률이 높은 발언과, 시장 대비 초과 성과를 낸 발언을 구분하여 보여줍니다.

### 3.3 어떤 전문가가 꾸준히 잘 맞췄는가?

일회성 우연이 아닌, 지속적으로 높은 적중률을 보이는 전문가를 식별합니다. 표본 수가 충분한 전문가 위주로 랭킹을 제공하여 통계적 신뢰성을 확보합니다.

### 3.4 특정 산업에서 누가 가장 강한가?

산업별로 특화된 전문가를 발굴합니다. 반도체, 2차전지, 바이오 등 특정 섹터에서 뛰어난 예측 능력을 보이는 전문가를 산업별로 확인할 수 있습니다.

### 3.5 지식을 많이 제공한 사람은 누구인가?

수익률과 무관하게, knowledge note 작성을 통해 커뮤니티에 지식을 기여한 사용자를 인정합니다. 지식 기여 랭킹은 커뮤니티 활성화와 참여 유도에 기여합니다.

## 4. 6가지 랭킹 종류

### 4.1 발언 수익률 TOP

발언 자체의 단순 수익률 기준 상위 순위를 보여줍니다.

| 필드 | 설명 |
|------|------|
| 순위 | 수익률 기준 순위 (1위부터 N위까지) |
| 발언자 | 발언을 한 전문가 이름 |
| 종목 | 발언 대상 종목명 |
| 산업 | 종목의 산업 분류 |
| 발언일 | 발언이 이루어진 날짜 |
| 검증기간 | 발언 검증에 사용된 기간 |
| 수익률 | 발언 후 실제 주가 수익률 (%) |
| 초과수익률 | 동기간 시장 수익률 대비 초과 수익률 (%) |
| 출처 | 발언 원본 출처 (뉴스, 방송, SNS 등) |

### 4.2 발언 초과수익률 TOP

벤치마크(코스피, 코스닥 등) 대비 초과수익률 기준 상위 순위를 보여줍니다. 시장 방향성과 무관하게 전문가의 실제 예측 능력을 평가하는 데 유용합니다.

| 필드 | 설명 |
|------|------|
| 순위 | 초과수익률 기준 순위 |
| 발언자 | 발언을 한 전문가 이름 |
| 종목 | 발언 대상 종목명 |
| 산업 | 종목의 산업 분류 |
| 발언일 | 발언이 이루어진 날짜 |
| 검증기간 | 발언 검증에 사용된 기간 |
| 발언 수익률 | 발언 후 실제 주가 수익률 (%) |
| 초과수익률 | 동기간 시장 수익률 대비 초과 수익률 (%) |
| 벤치마크 | 비교 대상 지수 |
| 출처 | 발언 원본 출처 |

### 4.3 전문가 평균 초과수익률 TOP

전문가의 전체 발언에 대한 평균 초과수익률 기준 순위입니다.

| 필드 | 설명 |
|------|------|
| 순위 | 평균 초과수익률 기준 순위 |
| 전문가 ID | 전문가 식별자 |
| 전문가명 | 전문가 이름 |
| 소속 | 전문가의 소속 기관 |
| 검증 발언 수 | 평가에 사용된 전체 발언 개수 |
| 평균 수익률 | 전체 발언의 평균 수익률 (%) |
| 평균 초과수익률 | 전체 발언의 평균 초과수익률 (%) |
| 최고 수익률 | 개별 발언 최고 수익률 |
| 최저 수익률 | 개별 발언 최저 수익률 |

> **참고**: 초기에는 검증 발언이 5개 이상인 전문가만 랭킹에 포함합니다. 이는 통계적 신뢰성을 확보하기 위한 최소 기준입니다.

### 4.4 전문가 적중률 TOP

전문가의 발언 적중률(Accuracy) 기준 순위입니다.

| 필드 | 설명 |
|------|------|
| 순위 | 적중률 기준 순위 |
| 전문가 ID | 전문가 식별자 |
| 전문가명 | 전문가 이름 |
| 소속 | 전문가의 소속 기관 |
| 검증 발언 수 | 평가에 사용된 전체 발언 개수 |
| 적중(Hit) 수 | 정확히 맞춘 발언 개수 |
| 부분 적중(Partial Hit) 수 | 일부만 맞춘 발언 개수 |
| 미적중(Miss) 수 | 틀린 발언 개수 |
| 적중률 | (Hit + 0.5 × Partial Hit) / 전체 발언 × 100 (%) |
| 평균 수익률 | 전체 발언의 평균 수익률 (%) |

> **적중 기준**: Hit = 1점, Partial Hit = 0.5점으로 계산합니다. 이는 발언이 완전히 맞았는지 부분적으로 맞았는지를 세분화하여 평가하기 위함입니다.

| 해당 산업 평균 수익률 | 특정 산업 발언의 평균 수익률 (%) |
| 해당 산업 적중률 | 특정 산업 발언의 적중률 (%) |

> 지식 기여 랭킹은 수익률과 무관하며, 커뮤니티 기여도에 초점을 맞춥니다.

## 5. 랭킹 상세 설계

### 5.1 발언 수익률 상세 설계

- **기준일**: 현재 시점 기준 최근 1년, 6개월, 3개월, 1개월 중 선택 가능
- **검증기간**: 발언 후 1일, 1주일, 1개월, 3개월, 6개월, 1년 중 선택
- **수익률 계산 방식**: (종가 - 기준가) / 기준가 × 100
- **기준가**: 발언일 종가
- **종가**: 검증기간 종료일 종가
- **제외 조건**: 상장 폐지 종목, 스플릿/리버스 스플릿 발생 종목은 별도 표시

### 5.2 초과수익률 상세 설계

- **벤치마크 설정**: KOSPI, KOSDAQ, S&P 500 중 선택 (시장 기본값은 KOSPI)
- **초과수익률 계산**: 발언 수익률 - 벤치마크 수익률
- **벤치마크 수익률**: 발언일과 동일 기간의 지수 수익률
- **음수 초과수익률**: 벤치마크를 하회한 경우 음수로 표시

### 5.3 전문가 평균 초과수익률

- **가중치 없음**: 모든 발언에 동일한 가중치 적용 (단순 평균)
- **표본 수 필터링**: 초기 5개 이상 발언 보유 전문가만 표시
- **이상치 제거**: 평균 ± 3 표준편차를 벗어난 발언은 이상치로 간주하여 제외 옵션 제공

### 5.4 적중률 계산 방식

- Hit (정확): 발언 방향과 실제 주가 방향이 일치하고, 변동폭이 예측 범위 내
- Partial Hit (부분 적중): 발언 방향은 맞았으나 변동폭이 예측 범위를 벗어남
- Miss (미적중): 발언 방향과 실제 주가 방향이 불일치
- **적중률 공식**: (Hit 개수 + 0.5 × Partial Hit 개수) / (Hit + Partial Hit + Miss 개수) × 100

### 5.5 산업별 랭킹

- 산업 분류는 KSIC (한국표준산업분류) 기준을 기본으로 하되, 사용자 친화적인 카테고리로 매핑
- 하나의 종목이 여러 산업에 속할 경우 주요 산업 기준으로 분류
- 산업별 발언 수가 3개 미만인 전문가는 해당 산업 랭킹에서 제외

### 5.6 지식 기여 랭킹

- Knowledge Note는 공개 상태인 것만 집계
- 삭제된 note, 비공개 note는 집계에서 제외
- 중복 작성 방지를 위해 동일 사용자의 유사 내용 note는 1개로 통합 집계

## 6. 표본 수 표시 원칙

모든 전문가 랭킹에서 표본 수(검증 발언 개수)를 반드시 표시합니다. 이는 사용자가 통계적 신뢰성을 스스로 판단할 수 있도록 하기 위함입니다.

### 6.1 표본 수 표시 예시

- 검증 발언 42개, 적중률 61.9%, 평균 수익률 +14.2%, 평균 초과수익률 +6.8%
- 검증 발언 5개, 적중률 80.0%, 평균 수익률 +5.1%, 평균 초과수익률 +2.3%

### 6.2 표본 수에 따른 시각적 구분

- **10개 이상**: 기본 표시 (신뢰성 높음)
- **5~9개**: 노란색 배지로 표시 (신뢰성 보통)
- **5개 미만**: 랭킹에서 제외 (초기 정책, 추후 변경 가능)

## 7. 랭킹 왜곡 방지

### 7.1 종합 점수 미도입

초기 버전에서는 종합 점수를 도입하지 않습니다. 6개 기준별로 각각의 랭킹을 제공하여, 하나의 숫자가 모든 것을 대표하는 오해를 방지합니다.

### 7.2 표본 수 최소 기준

- **전문가 평균 초과수익률**: 최소 5개 발언
- **전문가 적중률**: 최소 5개 발언
- **산업별 전문가 랭킹**: 해당 산업 최소 3개 발언
- **지식 기여 랭킹**: 최소 1개 knowledge note (기준 자체가 존재 여부)

### 7.3 극단값 처리

- 수익률 상/하위 5%는 이상치로 간주하여 별도 표시
- 사용자에게 극단값 포함/제외 옵션 제공
- 극단값 제외 시 랭킹에 영향이 있음을 사용자에게 고지

### 7.4 시간 가중치

- 최근 3개월 이내 발언에 가중치를 두는 옵션 검토 (추후 도입)
- 발언 시점으로부터 2년 이상 경과한 발언은 별도 표시

- **정렬 기준**: 각 탭의 기본 정렬 기준 외 사용자 선택 가능

### 8.4 리스트 항목 디자인

- 순위는 숫자로 표시 (1, 2, 3위는 강조 표시)
- 수익률은 양수/음수에 따라 색상 구분 (양수 빨간색, 음수 파란색)
- 전문가 이름 클릭 시 해당 전문가 상세 페이지로 이동
- 종목명 클릭 시 해당 종목 분석 페이지로 이동

### 8.5 반응형 디자인

- 데스크탑: 전체 테이블 표시 (최대 10개 컬럼)
- 태블릿: 주요 컬럼 축소 표시 (최대 6개 컬럼)
- 모바일: 카드 형태로 변환하여 표시

| 랭킹 종류 | 기본 정렬 기준 | 추가 정렬 옵션 |
|-----------|---------------|----------------|

| 발언 초과수익률 TOP | 초과수익률 (내림차순) | 수익률, 발언일 |

| 전문가 적중률 TOP | 적중률 (내림차순) | 발언 수, 평균 수익률 |
| 산업별 전문가 랭킹 | 평균 수익률 (내림차순) | 적중률, 발언 수 |
